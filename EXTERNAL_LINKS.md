# External Community Patch Links

External community patches are linked only.

Their source code is not copied into this repository.

## Rules

When adding an external patch link:

1. Do not copy its source code into this repository.
2. Keep the patch author's name if known.
3. Link to the original patch page, fork, release, or commit.
4. Do not claim the external fix as your own.
5. If the author asks for the link to be changed or removed, update or remove it.
