Unofficial ADOFAI mod patch hub maintained by Building114.

This repository lists temporary compatibility patches for ADOFAI mods.

Some entries are maintained by Building114.  
Some entries may only be external links to community patches.

External community patches are linked only. Their source code is not copied into this repository.

## Important Notice

- This repository is not an official project.
- I am not the original author of the listed mods unless clearly stated otherwise.
- Original projects belong to their original authors and contributors.
- Not all original projects listed here are necessarily archived.
- External community patches follow their own repositories' rules and licenses.
- If an original author or patch author wants a link corrected or removed, I will respect that request.
