The original TimingShow README credited these projects as references:

- Overlayer: https://github.com/c3nb/Overlayer
- ADOFAI ProgressDisplayer2: https://github.com/FLOWERs-Modding/ADOFAI_ProgressDisplayer2

At the time this notice was prepared, both referenced repositories were visible on GitHub, but their repository pages did not show an obvious root license file in the file list. Because of that, this patch package treats them as references only.

This cleaned source package does not include those projects as standalone copied source trees, and it does not include copied third-party binaries from them.

Code from Overlayer or ADOFAI ProgressDisplayer2 is not treated as included source here unless a future patch clearly adds the matching license and attribution.

## Local Build Dependencies

The project needs local ADOFAI assemblies, Unity assemblies, Harmony, and UnityModManager assemblies to compile. Those dependencies are not included in this source package.

Private game paths, copied game assemblies, UnityModManager binaries, Harmony binaries, and restored local build output are outside the scope of this source package.
