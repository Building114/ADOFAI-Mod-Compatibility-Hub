This repository is an unofficial compatibility hub for ADOFAI mods.

The original mods belong to their original authors and contributors.

Unless clearly stated otherwise:

- I am not the original author of the listed mods.
- This repository is not an official release from the original mod authors.
- Building114-maintained entries are temporary compatibility patches.
- External community patches are linked only.
- External community patch source code is not copied into this repository.
- Original project names are used only to identify what each patch is related to.
- Not all original projects listed here are necessarily archived. Check each entry separately.
- Users should prefer official updates if they become available.

If an original author or external patch author wants a link corrected or removed, please open an issue or contact me through GitHub.
